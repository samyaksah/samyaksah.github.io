CONTEXT

About Splitwise

Splitwise is a mobile-first platform that helps people in group activities split their bills easier reducing the stress of doing confusing and on-the-spot maths. The app eliminates the need to keep receipts and the need to keep tabs on individual expenses that involve the user(borrowing or lending). The user can add expense to the app as soon as the cost is incurred. For a large group, it can be difficult to share expenses but splitwise makes the decision process a fair.

RESEARCH PROCESS

Understanding Audience

Splitwise We focused down our demographic to university students who want to be responsible with their money. For the purposes of creating a persona, we chose two roommates as our ideal demographic.

To see what makes Splitwise special and what can be improved we needed to understand how the app works. We conducted several think-aloud tests participants that were part of our target demographics.

Problem with the app

View feature introduction slides here.
From our research, my team-mate and I found a few major problems with the app which needed to be addressed.

Honour System “Settle Up”

In the app, the SETTLE UP button is visible to both the lender and the borrower. Because of this, the borrower can “settle up” without actually paying the lender. And since Splitwise does not have an in-built online transaction to directly transfer money, the lack of this feature can cause major misunderstandings or complications between users.

Reminders

The current reminder functionality of Splitwise sends either a text or an email reminder. The functionality fails for users without cellular devices and with emails, the reminders get lost in updates which tend to be ignored by the users.

Splitwise is a Collaboration Tool

Splitwise is a computer-supported cooperative work (CSCW) system that falls under the communication + coordination quadrant in the Time/Space Groupware matrix( i.e. users can collaborate at different places and at different times). Another collaboration tool that falls under the same category is Slack which has a very clean and structured chat system supported by bots that helps users with file housekeeping and other basic tasks. Which inspired us to implement a handy chat feature in our app that would work as a central hub for all communications around finances on Splitwise.

Splitwise is primarily a Communiction + Coordination (quad 4) software

FEATURE BREAKDOWN

wiseBOT

Initial Interaction

There are two tabs inside the chat page. An individual chat and a group chat. The individual chat contains conversations between two users, where as all the groups have a chat in the group tab. There also exists a wiseBot tab on the top of the page. The placement of the tab is fixed on the top. to allow easy access to the bot.

Within Users Interaction

A new chat between two users is created at any instance of interaction between them on the app. The chat also works as a mini timeline for all the transactions that concern the two users. Each chat also comes with the wiseBot that handles all of the app’s features and can be accessed by “.bot/(enter command)”

User and wiseBot Interaction

A wise bot implementation in the chat where the user can find all the documentation about the application. To demonstrate the chat feature with wisebot, there will be a short and detailed on-boarding process which gets the user familiar with the update. The wisebot allows the user to operate and use the applications functionality from the chat and suggests automation for repetitive tasks.
